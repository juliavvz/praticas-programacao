Este repositório serve como um espaço dedicado aos meus estudos e aprimoramento em lógica de programação e desenvolvimento de sistemas.
Aqui, você encontrará uma coleção de programas simples, exercícios e pequenos projetos que estou desenvolvendo para consolidar meus 
conhecimentos e praticar diferentes conceitos de programação.

🎯 O principal objetivo deste repositório é:

- Praticar e aprimorar minhas habilidades em diversas linguagens de programação.
- Explorar e implementar algoritmos e estruturas de dados básicas.
- Documentar meu progresso e aprendizado ao longo do curso de Análise e Desenvolvimento de Sistemas.
- Servir como um portfólio de meus primeiros passos na programação, demonstrando minha dedicação e evolução.

🛠️ Os projetos e exercícios aqui presentes são desenvolvidos principalmente com as seguintes tecnologias:

- Python: Para scripts de automação, lógica e manipulação de dados.
- TSQL (Transact-SQL): Para exercícios e consultas em bancos de dados SQL Server.
- Outras linguagens e ferramentas serão adicionadas conforme meu aprendizado avançar.

📂 Estrutura do Repositório

O repositório está organizado em pastas, onde cada uma representa uma linguagem, um tópico específico ou um conjunto de exercícios.

▶️ Como Executar os Projetos

Para executar os programas e scripts deste repositório, siga as instruções abaixo, adaptando-as à linguagem e ao ambiente necessário:

Python:

1. Certifique-se de ter o Python instalado em sua máquina.
2. Navegue até a pasta do projeto Python desejado (ex: cd python/).
3. Execute o script usando o interpretador Python:

BASH
python nome_do_arquivo.py

TSQL:

1. Certifique-se de ter acesso a um ambiente com SQL Server (local ou remoto).
2. Utilize uma ferramenta como SQL Server Management Studio (SSMS) ou Azure Data Studio.
3. Abra o arquivo .sql desejado e execute as consultas em seu banco de dados.
